How to Use This Page
  • ELEMENTS — Click on any element to see details
  • PERIODS — Click any button on the column on the left to learn about each horizontal row of the table
  • GROUPS — Click any button above to explore that element families and see all its members
  • Hover over any element cell to see its full name, category, and atomic number
  • LANTHANIDES (57–71) and ACTINIDES (89–103) — follow the pointing arrows to rows below the table
  • Elements with RED symbols are synthetic — produced only in laboratories, never found in nature
